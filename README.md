# Explorando IA Generativa em um Pipeline ETL com Python

Este repositório demonstra como **integrar técnicas de IA generativa em um pipeline de ETL (Extract, Transform, Load)** utilizando Python.  
O objetivo é explorar como modelos de IA podem enriquecer processos de extração, transformação e carregamento de dados em cenários reais de negócios.

---

## 📌 Objetivos do Projeto
- Implementar um pipeline ETL simples em Python.
- Explorar como IA generativa pode ser aplicada em etapas de transformação de dados.
- Demonstrar boas práticas de organização de código para projetos de ciência de dados e engenharia de dados.
- Servir como base de estudo para profissionais e estudantes interessados em IA aplicada a ETL.

---

## 📂 Estrutura do Repositório
- `explorando_ia_generativa_pipeline_etl_python.py` → Script principal que contém a implementação do pipeline ETL com integração de IA generativa.
